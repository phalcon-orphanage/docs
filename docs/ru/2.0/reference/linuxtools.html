Инструменты разработчика Phalcon Developer Tools в Linux¶

Этот шаг поможет вам установить Phalcon Developer Tools в Linux.

Требования¶

Для запуска инструментов разработчика необходимо установленное расширение Phalcon PHP. Если оно еще не установлено, прочитайте инструкции в разделе Installation.

Скачать¶

Вы можете скачать кроссплатформенный пакет инструментов разработчиков из раздела Download. Также вы можете его слить ( git clone ) на Github. Введите в терминале следующие команды:

Поздравляем, инструменты разработчика Phalcon установлены!
