Class Phalcon\Di\Service¶

implements Phalcon\Di\ServiceInterface

Source on GitHub

Represents individually a service in the services container

<?php

$service = new \Phalcon\Di\Service(
    "request",
    "Phalcon\\Http\\Request"
);

$request = service->resolve();

<?php
