Class Phalcon\Db\Adapter\Pdo\Postgresql¶

extends Phalcon\Db\Adapter\Pdo

Specific functions for the Postgresql database system

<?php

 $config = array(
  "host" => "192.168.0.11",
  "dbname" => "blog",
  "username" => "postgres",
  "password" => ""
 );

 $connection = new Phalcon\Db\Adapter\Pdo\Postgresql($config);
